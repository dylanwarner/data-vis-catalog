Cluster

Summary

A cluster diagram,or dendrogram, is a hierarchichal node-link chart where leaf nodes of the tree are on the same level. Cluster diagrams are useful for hierarchichal data or data that involves groupings.

Analysis

Cluster diagrams use lines and points (circles) as marks. These are represented by the nodes and connection lines. These marks help to create a sense of hierarchy and relationships between nodes. Color can be used as a channel to group nodes or represent depth.
