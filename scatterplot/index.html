Scatterplot

Summary

A scatterplot uses coordinates to map values of two different variables from a dataset. A scatterplot uses a collection of circle points with coordinates mapping to the horizontal and vertical axes. Scatterplots are useful when comparing two data variables in a dataset.

Analysis

Scatterplots use points as marks. Vertical and horizontal positioning are used to place data points. Circle size (radius) can be mapped to a variable to visually show higher and lower values. Color can also be used to group data points or separate them.
