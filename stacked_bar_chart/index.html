Stacked Bar Chart

Summary

A stacked bar chart is like a bar chart, except the data is stacked on top of each other to show how the overall dataset is divided into categories. A stacked bar chart is useful when multiple variables in a dataset need to be compared at the same time.

Analysis

A stacked bar chart uses lines and areas as marks. An important channel for stacked bar charts is color. Color is used to represent the data variables and split the bars into different sections.
